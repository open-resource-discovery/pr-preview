Entities with Associations

Description​

This example focuses on the use of cds.Association and the @ObjectModel.foreignKey.association annotation.

Example File: entities_with_foreign_key_and_text_assocs​

          "length": 3

        "texts": {

            "max": "*"

              "ref": ["code"]

      }

        },
        "descr": {
          "type": "cds.String",
          "length": 1000
        }
      }
    },
    "FlightConnection": {
      "kind": "entity",
      "elements": {
